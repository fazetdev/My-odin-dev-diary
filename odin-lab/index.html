This space is a personal archive of all the projects I’ve built while following The Odin Project, a full-stack web development curriculum designed to take learners from zero to developer.

Each project here reflects a specific lesson, concept, or skill I’ve learned — from basic HTML and CSS to JavaScript, Git, and beyond. Whether it’s creating a recipe website, building a landing page, or learning to style layouts with Flexbox, every step represents growth.

The lab is organized to help me track my progress, share my work, and stay motivated as I move forward in my web development journey.

Feel free to explore each project, read the code, and see how my skills improve over time. This is more than just a collection of files — it’s a timeline of learning, practice, and discovery.
