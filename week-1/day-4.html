🔴 Day 4 – The Odin Recipes Project Begins

Today I officially started my first real Odin project: Odin Recipes. After learning how to write solid commit messages, I created the index page and began adding recipe pages for some of my favorite Hausa dishes — Suya (Tsire), Wainar Shinkafa, and Dambun Nama. Seeing my culture represented in my code felt deeply motivating. It wasn’t just “HTML practice” anymore — it was personal.

The big challenge wasn’t writing HTML — it was moving files between my laptop and phone. I ran into issues pushing and pulling from GitHub using both the laptop terminal and Termux. Fetching and syncing code between devices was confusing at first.

But I discovered something useful: it’s easier to move files from Termux to the Micro editor and manage things there, than trying to fetch files directly from GitHub to VS Code on the laptop.

These small victories made me realize: knowing how to move your code around is as important as writing it.

I also updated my Odin Dev Diary with today’s progress. While I haven’t finished the full project yet, I’ve made solid ground. The recipes are showing up, the structure is in place, and Git is slowly becoming less scary.

I’m learning by doing, just as I promised.
Onward.
