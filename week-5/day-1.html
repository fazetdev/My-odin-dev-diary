    • 🧠 What I Learned

    I explored the idea of Clean Code — not just making the program work, but making it readable, understandable, and maintainable. I revisited old code to improve variable names, remove redundancy, and simplify logic. I learned that code should be written for humans first, machines second.

    🧪 Experiments

    I practiced rewriting old functions and loops to be cleaner and easier to follow. I added meaningful comments and tested renaming variables to express intent clearly.

    ❌ Problems

    Some of the resources felt repetitive. Concepts overlapped, and switching between multiple sources made it feel slightly boring. It was difficult staying focused at some point.

    🌬️ Wind (Motivation)

    The idea that I’m not just learning to code — but to write something others can understand — gave me purpose. I want to be someone whose code is praised for being elegant, not just functional.
